深度学习笔记.1-基础知识

发表于 2024-11-07 分类于 AI

智能范畴

Artificial Intelligence, AI

各种模拟智能体行为的系统

Machine Learning

基于数学模型来拟合观测数据，以做出决策

Deep Learning

基于深度神经网络模型来拟合数据的机器学习系统

机器学习方法

有监督学习 - supervised learning

基于有人工标签的训练数据，进行模型训练的过程。通过标签，模型学习了输入-输出的映射关系。

无监督学习 - unsupervised learning

基于没有人工标签的训练数据，进行模型训练的过程。模型最终描述了数据中包含的隐藏结构和关系。

生成式无监督学习模型 - generative unsupervised models

用于生成新的数据，新数据和训练数据在统计特征上保持一致。

潜在变量 - latent variable

将原始数据映射为底层的潜在变量，从而可以降低数据维度，并更简单的描述其统计分布特征。

联合监督和无监督学习

通过无监督学习的潜在变量，可以将监督学习中，学习训练集的输入数据和输出数据的关系，转换为学习输入和输出的潜在变量的映射关系，从而可以获得更好的泛化结果。

强化学习 - reinforcement learning

需要进行一系列连续决策，以形成最优策略，获得在统计特征上最好的最终结果。
环境可以定义为一系列时间/空间连续的状态机；
信息输入不是初始完备的，会随着决策持续更新；
需要作出执行已知最佳方案和探索新方案的选择；
如机器人运动学习、棋类对抗。
深度神经网络可用于构建从现实信息输入到行为决策的策略网络。

决策类型

回归问题 - Regression

用一个连续值来表示决策

多元回归问题 - Multivariate Regression

结果同时预测了多个连续值，

分类问题 - Classification

二元分类问题 - Binary Classification

将输入归纳于两个分类选项的一个

多元分类问题 - Multiclass Classification

将输入归纳与多个分类选项中的其中一个
